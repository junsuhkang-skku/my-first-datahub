# 🎮 Gaming Dashboard

A simple and interactive **Gaming Stats Dashboard** built with **Streamlit**, **Pandas**, and **Plotly**.  
Track your game performance, visualize stats, and monitor progress in a clean UI.

---

## 📌 Description

This project is a web-based dashboard that allows users to:

- Track wins, losses, and playtime
- Calculate performance metrics like win rate and rank score
- Visualize gaming data through interactive charts
- Add new game data dynamically via sidebar input

It’s designed for gamers who want a quick overview of their performance across different games.

---

## ✨ Features

- 🎮 Add and manage multiple games
- 📊 Automatic calculation of:
  - Win Rate
  - Total Matches
  - Rank Score
  - Weekly Play Time
- 📈 Interactive visualizations:
  - Wins vs Losses (Bar Chart)
  - Weekly Play Time (Line Chart)
  - Rank Progression
- 🎯 Game-specific filtering
- ⚡ Real-time updates using Streamlit session state
- 🟢🔴 Performance indicator (based on win rate)
